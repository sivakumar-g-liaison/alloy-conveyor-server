A Seed for serious web apps...

This project is meant to serve as a feature-rich starting point for enterprise and commercial web applications. It integrates Netflix-karyon and AngularJS with projects like Angular-UI, Angular-Strap, Bootstrap, and jQuery, along with examples of angular services, directives, controllers, and filters. And adds aditional support for auditing, dynamic web services, and more.

Integrated Technologies:

  • karyon - The seed for webservices used within the NetFlix infrastructure.
  • angularjs - Superheroic JavaScript MVW Framework.
  • angular-ui - The companion suite for AngularJS.
  • Twitter Bootstrap - Sleek, intuitive, and powerful front-end framework for faster and easier web development.
  • angular-strap - Bootstrap directives for Angular.
  • ng-grid - Angular Data Grid.
  • jquery - The Write Less, Do More, JavaScript Library.
  • jquery-ui - The Write Less, Do More, JavaScript Library.

  • prettifyjs - A Javascript module and CSS file that allows syntax highlighting of source code snippets in an html page.
  • Animate.css - Just-add-water CSS animation. via Ad Packs. animate.css is a bunch of cool, fun, and cross-browser animations for you to use in your projects.

Project goals:

  • A Java-based, container agnostic webservice seed
  • Jersey integration for easy RESTFul webservice implementations, with working examples
  • Deploy-independent administrative console (located at {host}:8077)
  • Styling via Bootstrap with ready-to-build LESS scripts
  • Integrate best-of-breed JavaScript MVC, styling, ui, and tooling components "the Angular way"
  • Provide examples of services, directives, controllers, and filters in a way that can be cherry picked for use elsewhere.
  • See Authentication and REST service examples and fadein and stop-propagation directive examples.
  • Provide example of ui widgets done the Angular way. See the Playground menu option in the top navbar. UI/widget examples include a paginated grid with sort and filter support, accordion, tree, tabs, and date/time pickers.
